# CRM Lead Management System

Система управления лидами — CRM для холодных звонков с поддержкой ИНН и СВО баз.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Сервер запустится на **http://localhost:3000**

## Вход

- **Логин:** `admin`
- **Пароль:** `admin123`

## Основные функции

- 📞 Менеджмент лидов — карточки, звонки, передачи
- 👥 Управление работниками — создание, роли, доступы  
- 📊 Аналитика — статистика по дням, KPI, ТОП-10
- 📂 Импорт баз из Excel/CSV
- 🏢 Департаменты — ИНН и СВО
- 🎰 Рулетка и талисманы за монеты (BorodaCoins)
- 🌙 Тёмная тема, кастомизация интерфейса

## Структура

```
├── server.js          — Сервер (Express.js)
├── database.js        — JSON-база данных
├── package.json       — Зависимости
├── data/              — Данные (JSON файлы)
├── public/            — Клиентская часть
│   ├── index.html     — Главная страница
│   ├── app.js         — Логика приложения
│   ├── styles.css     — Основные стили
│   ├── themes.css     — Темы оформления
│   ├── dept-v2.css    — Стили департаментов
│   └── mascots/       — Талисманы
├── uploads/           — Загруженные файлы
└── files/             — Файлы системы
```
